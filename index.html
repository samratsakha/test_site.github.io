How to get exact resale value..?



Model


Select your Apple iPhone Model from the list


Variant


Select your Apple iPhone variant from the list


Physical Damage


Select Yes if your iPhone has any Physical Damages

Example: Screen damage/Touch Cracks

Negligible scratches/dents doesn't mention any damage


Software Issues


Select Yes if your iPhone has any Software Issues

Example: Call Network problems/Phone not Starting


Hardware Damage


Select Yes if your iPhone has any Hardware Issues

Example: Touch ID/Face ID not working


Battery health


Select Battery Health of your iPhone

Settings -> Battery -> Battery Health


Kit Available or not


Select Yes if you are having your iPhone's Kit or else No

Kit includes -> iPhone's Box,Lightining Cable,Adapter,Earphones


Hit the Calculate Button
